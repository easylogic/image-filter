# simple bitmap image filter With EasyLogic


# Install 

## npm 

```npm
npm install @easylogic/image-filter
```

# How to use 
 
```
import ImageFilter from '@easylogic/image-filter' 
```


# Developments 

## local dev 

```
git clone https://github.com/easylogic/image-filter
cd image-filter
npm install 
npm run dev 
open localhost:3000
```

## build 

```
npm run build 
```
